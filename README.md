# PREDEXAN

## Welcome to the repository

**Status: Project temporarily on hold**

### 1. About the project

Predexan is a Python package for Data Science and Machine Learning Engineering. 
 
It started in an [university project](https://github.com/rafaeldecastro195/aprendizado-supervisionado-cybersecurity.git), in 7th semester of Bachelor in Information Systems at PUC Minas, about `Supervised Learning` for cyber attacks,  using the dataset [UNSW-NB15](https://www.kaggle.com/datasets/dhoogla/unswnb15).  

As the project grew in complexity, the need to develop a package became very clear. There were lots of functions for plotting graphs and, specially, printing metrics.

Developing this package was one of the most important works of all the project and I assume that it might be considered another project as well. For that, I decided to carry on with that package and make it more usable and reliable in different contexts and use cases of `Exploratory Data Analysis`, `Data Preprocessing` and `Supervised Learning`.

### 2. Features

For now, the package has 4 modules: `metrics`, `graphs`, `preproc` and `deploy`. My goal is to create a module named `pred` with classes and functions for the predictive models.

All of the mentioned modules already have functions and deploy has a class. More code and testing will be added.

### 3. Next steps

The very first step taken will be to translate all the content from Brazilian Portuguese to English, including names of modules and functions, graphs titles, axis labels and texts from metrics.

After that, the focus will be implementing tests and QA, improving code quality, adding new features, and potentially introducing other modules.
